# **App Name**: ClubConnect

## Core Features:

- Family Configuration: Configure family member profiles with names and optional photos.
- Check-in Feature: Select family members heading to the club, destination(s) within the club, and planned duration of stay.
- Update Location: Update a family's current location to specify where the kids will be.
- Location Dashboard: Display real-time view of club locations and the families checked in at each.
- Club Customization: Configure basic branding, configurable club name, club logo
- Notification Preferences: Allow subscribed users to pick which families they want to get notified for when they head to the club
- Push Notifications: Push notifications sent to subscribers when chosen families check in.

## Style Guidelines:

- Primary color: Sky Blue (#87CEEB), evoking a sense of calm and community suitable for family activities.
- Background color: Light gray (#F0F0F0), providing a clean and neutral backdrop.
- Accent color: Coral (#FF7F50), used to highlight key interactive elements and calls to action.
- Body and headline font: 'PT Sans' (sans-serif), which is modern but still offers a little warmth for a family focused application
- Simple, recognizable icons representing different club locations.
- Clean and intuitive layout for easy navigation and clear presentation of information.
- Subtle transitions when families check-in, check out, or leave comments, giving it a natural feel
